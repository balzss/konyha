# Konyha Recipe Manager

Self hostable recipe manager written in typescript.

### Prerequisites

- Docker and Docker Compose
- Node

### Setting it up:

Install dependencies:

```
yarn install
```

- set env vars
- start db
- push to db
- generate graphql

### Running it:

Start the database:

```
yarn postgres
```

And start the client:

```
yarn dev
```

### Development:

- updating the database schema
- updating the graphql schema
